moment-locales

build-plugin-moment-locales 将对 moment 依赖的使用进行优化：

  • 优化 moment 语言包加载
  • 根据设置自动加载对应语言包

Install#

$ npm i --save-dev build-plugin-moment-locales

Options#

  • locales：类型 String | Array，需要加载的多语言包

Usage#

{
"plugins": [
[
"build-plugin-moment-locales",
{
"locales": ["zh-cn", "en-au"]
}
]
]
}
